  • convertToPixel(value:orientation:)

    This function convert the value to the pixel. If the value has % at the end, it converts the value to the pixel value based on the orientation.

    Declaration

    Swift

    public static func convertToPixel(value : String, orientation : Orientation?) -> Double

    Parameters

    value

    For pixel values just pass the number. For percentage based value pass the % at the end of the number.
    orientation

    For top and bottom it is Vertical and left and right it is Horizontal.

    Return Value

    Returns the pixel based value. Returns null if the canvas height or width is not defined.
